PyPathlen.

A Python pathlength checker

Github

What is PyPathlen?

PyPathlen is a script written in Python 2.7. It creates a report for a given folder location:
  • Listing all files and folders plus their pathlength
  • Comparing the pathlength with 2 adjustable tresholds (Alert and Fail)
  • Marking the pathnames with [OK], [ALERT] or [FAIL]

What are the benefits?

Some OS have character length restrictions resulting in shortened filenames or error messages.

With PyPathlen you can avoid such bad "surprises".

How to use PyPathlen?

Just run python pathlen.py -h to get an overview of the available options. Have a look at following sample:

$ python pathlen.py -p ./ -a 20 -f 50
